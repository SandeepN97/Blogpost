# Blog Application

A simple blog application built with Spring Boot.

## Features

- User registration and login
- Create, edit, delete, and view blog posts
- Comment on blog posts
- View user profiles
- Like and unlike posts
- View top liked posts

## Technologies Used

- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL database
- MySQL (for production)
- Spring Security
